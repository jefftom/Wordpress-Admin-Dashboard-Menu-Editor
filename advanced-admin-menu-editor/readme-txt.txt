Customize your WordPress admin menu with support for multiple configurations and theme customization.

== Description ==

Advanced Admin Menu Editor allows you to completely customize the WordPress admin menu to meet your specific needs. Create different menu configurations for different user roles, rename menu items, hide unused options, and even customize the appearance of the admin menu.

= Key Features =

* **Multiple Configurations** - Create and save different menu configurations.
* **Role-based Assignments** - Assign different menu configurations to different user roles.
* **Drag and Drop Interface** - Easily reorder menu items with a simple drag and drop interface.
* **Theme Customizer** - Customize the appearance of your admin menu with color schemes, spacing, and more.
* **Google Fonts Integration** - Choose from popular Google Fonts to enhance the typography of your admin dashboard.
* **Item-specific Permissions** - Control which roles can see specific menu items.
* **Rename Items** - Make menu items more intuitive for your users by customizing their names.
* **Hide Unused Items** - Simplify the admin interface by hiding menu items you don't need.

= Use Cases =

* Create a simplified admin menu for clients or editors
* Customize admin menus based on department or role needs
* Create a more intuitive interface for your team
* Adjust the admin menu to match your branding
* Hide advanced features from basic users

= Pro Tips =

* Create a separate configuration for administrators with full access
* Use the theme customizer to match your brand colors
* Hide rarely used items to create a cleaner interface
* Changes take effect immediately after saving

== Installation ==

1. Upload the `advanced-admin-menu-editor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Menu Editor' in your admin menu to start customizing

== Frequently Asked Questions ==

= Will this plugin affect my site's front end? =

No, this plugin only modifies the appearance and structure of the admin dashboard menu. It doesn't affect your website's front end or theme in any way.

= Can I create different menu configurations for different user roles? =

Yes! That's one of the main features of this plugin. You can create multiple menu configurations and assign them to specific user roles.

= What happens if I deactivate the plugin? =

When deactivated, your WordPress admin menu will return to its default state. Your configurations will be saved in the database and will be available if you reactivate the plugin.

= Is it compatible with other admin plugins? =

The plugin is designed to be compatible with standard WordPress admin interfaces. However, some plugins that heavily modify the admin area might have compatibility issues.

= Can I reset the menu to its default state? =

Yes, there's a "Reset to Default" button that allows you to restore the original WordPress menu structure.

= Can I export my menu configurations? =

This feature is planned for a future update.
